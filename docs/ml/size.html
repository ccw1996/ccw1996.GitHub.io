图像大小

## 卷积层
    O=输出图像的尺寸。
    I=输入图像的尺寸。
    K=卷积层的核尺寸
    N=核数量
    S=移动步长
    P =填充数
    ### 输出图像尺寸的公式：

$$O=(I-K+2P)/2+1$$ 输出图像的通道数等于核数量N 例：AlexNet中输入图像的尺寸为2272273.第一个卷积层有96个尺寸为11113的核。步长为4，填充为0. 输出的图像为555596（每个核对应1个通道）。

## 池化层
    O=输出图像的尺寸。
    I=输入图像的尺寸。  
    S=移动步长  
    PS=池化层尺寸  
    ### 输出图像尺寸的公式：

$$O=(I-Ps)/S+1$$ 不同于卷积层，池化层的输出通道数不改变。 例：每1层卷积层后的池化层的池化层尺寸为33，步长为2。根据前面卷积层的输出为555596。 输出尺寸为2727*96。

## 全连接层
    全连接层输出向量长度等于神经元的数量。

参数量

## 卷积层
    WC=卷积层的weights数量
    BC=卷积层的biases数量
    PC=所有参数的数量
    K=核尺寸
    N=核数量
    C =输入图像通道数
    ### 输出参数公式
        卷积层中,核的深度等于输入图像的通道数.于是每个核有K*K个参数.并且有N个核
        $Wv=K*K*C*N$
        $Bv=N$
        $Pv=Wv+Bv$
        例：AlexNet网络中,第1个卷积层,输入图像的通道数(C)是3,核尺寸(K)是11*11,核数量是96
            Wv=11*11*3*96=34848
            Bv=96
            Pv=34848+96=34944
## 池化层
    没有与MaxPool layer相关的参数量.尺寸,步长和填充数都是超参数.
## 全连接层
    连接到Conv Layer
        Wcf= weights的数量
        Bcf= biases的数量
        O= 前卷积层的输出图像的尺寸
        N = 前卷积层的核数量
        F = 全连接层的神经元数量
        ### 输出参数公式
            $Wvf=O*O*N*F$
            $Bvf=F$
            $Pvf=Wvf+Bvf$
            例：AlexNet网络中第1个FC层连接至Conv Layer.该层的O为6,N为256,F为4096.
                Wvf=6*6*256*4096=37748736
                Bvf=4096
                Pvf=Wvf+Bvf=37752832
    连接到FC Layer
        Wff= weights的数量
        Bff= biases的数量
        Pff= 总参数的数量
        F= 当前FC层的神经元数量
        F1 = 前FC层的神经元数量
            ### 输出参数公式
                $Wff=F1*F$
                $Bff=F$
                $Pff=Wff+Bff$
                例：AlexNet的最后1个全连接层,   F-1=4096,F=1000
                    Wff=4096*1000=4096000
                    Bff=1000
                    Pff=Wff+Bff=4097000

计算量

一般来说，将一个长度为I的向量与一个I×J的矩阵相乘，得到一个长度为J的向量，需要I×J的MACC或（2I-1）×J的FLOPS。
## 全连接层
    I输出层，J输出层(多batch得乘以batch)
        I*J MACs
        (2I-1)*J FLOPs （不考虑bias时有-1，有bias时没有-1。）
## 激活函数
    Relu
        J个输出元，JFLOPs
    Sigmod
        J*4 FLOPs
## LSTM
    实际上LSTM里面有 4 个非线性变换（3 个 门 + 1 个 tanh），每一个非线性变换说白了就是一个全连接网络
    维度就是embedding_size + hidden_size，第二层就是输出层，维度为 hidden_size，则它的计算量按照上文我们对全连接层的阐述，易得MACC为：
    `(embedding_size + hidden_size) * hidden_size * 4`
    `(embedding_size + hidden_size) * hidden_size * 4 个MACC`
    `embedding_size * hidden_size * 8 + hidden_size * (hidden_size + 20) 个FLOPS`  
    参数量:
        `((embedding_size + hidden_size) * hidden_size + hidden_size) * 4`
## 卷积层
    MACC:  $K*K*Cin*Hout*Wout*Cout$
        stride如果不是1，那输出特征图便减少，MACC也会减少
    FLOPs: $(2*Kw*Kh*Cin/g-1)*Cout*Hout*Wout$
        若有bias，不需要-1，g为分组卷积的分组数
## 深度可分离卷积
    深度卷积：
        MACC: $K*K*Cin*Hout*Wout$
    逐点卷积：
        MACC： $Cin*Hout*Wout*Cout$
    总计算：
        MACC： $Cin*Hout*Wout*(K*K+Cout)$
    MobileNetV2
        这种扩展块中MACC数量的公式：Cexp = (Cin × expansion_factor)，（expansion_factor用于创建深度层要处理的额外通道，使得Cexp在此块内使用的通道数量）

## **池化层**
    max、sum、avg pooling
        FLOPs： $Hin*Win*Cin$
    一般的池化
        FLOPs： $K*K*Hout*Wout*Cout$
